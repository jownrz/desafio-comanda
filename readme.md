# Desafio sobre Lógica de Programação e Pull Requests


## Como realizar o desafio:

1- Fazer um Fork desse repositório.  
2- Clonar o repositório para seu ambiente local.  
3- Criar uma branch para realizar o desafio.  
4- Criar uma pasta na raíz do desafio com seu nome (Ex: aldobraneluz).  
5- Criar um arquivo js dentro dela (Ex: desafio.js).  
6- Copiar o objeto "comanda" para realizar o desafio no seu arquivo.  
7- Após realizar o desafio, subir a branch pro github com o desafio.  
8- Solicitar o Pull-Request nesse repositório.  


## Simulação de tarefa de programação.

Nossa empresa foi encarregada de escrever um código para simular um PDV de um bar, seu trabalho foi definido para escrever o cálculo final da comanda. Para as regras do negócio exigiram também 2 valores finais, o valor dos produtos e um valor com o acrescimo de 10% pelo serviço para os que desejarem pagar o adicional.
